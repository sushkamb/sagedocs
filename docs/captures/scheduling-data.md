# Calendar & Scheduling — Captured UI Data

## How to Access
Click the "Calendar" tab in the top navigation.

## Calendar Views
The calendar toolbar has these view buttons:
- **Day** — Shows time slots (5-minute increments from 7:00 AM to 7:55 PM) across all rooms as columns
- **Week** — Week view with days as columns
- **Work Week** — Mon-Fri only
- **Agenda** — List view of appointments

## Toolbar Controls
- **Export to PDF** button
- **Today** button — Jump to today's date
- **Previous / Next** arrows — Navigate dates
- **Date display** button — Shows current date (e.g., "Thursday, February 12, 2026"), click to pick a date
- **Search Appointments** button — Search across all appointments
- **No Show List** button — View patients who didn't show up
- **Report** button — Generate appointment reports
- **Show full day** button — Expand to show all hours (not just business hours)

## Room Sidebar (Left Panel)
- Header: "Group By Rooms"
- Checkboxes to show/hide rooms:
  - 11042024
  - new rooms
  - Testing
  - Accupuncture
  - Massage Therapy
  - Treatment Room 1
  - Treatment Room 4
  - Treatment Room 5
- Each room appears as a column in the Day view
- Rooms can be configured in Administration > Calendar Settings

## Booking an Appointment
1. Click on an empty time slot in the calendar (intersect a time and room)
2. An appointment creation dialog opens
3. Fill in: Patient (search), Appointment Type, Provider/Doctor, Duration
4. Save the appointment

## Rescheduling
- Drag and drop an appointment to a new time slot or room
- Or click the appointment to edit and change the time

## Appointment Colors/Status
- Appointments appear as colored blocks in the calendar
- Color indicates status (checked in, confirmed, etc.)

## Quick Access
- "Appointments" link in the sub-navigation bar (below main tabs) provides quick access to today's appointments
